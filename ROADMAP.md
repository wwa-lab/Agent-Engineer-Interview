# Roadmap

The roadmap is a set of useful next slices, not a promise that every framework or provider will be covered.

## Available now

- Core English handbook with Chinese navigation.
- Beginner, intermediate, senior, and behavioral question banks.
- Five offline Python coding challenges.
- Agent loops, workflow boundaries, MCP, evaluation, system design, resume, and storytelling guides.
- Markdown, link, structure, and challenge test checks.

## Next

- Add worked solutions for more retrieval and security incidents.
- Add a staff-level rubric for architecture, influence, and operational ownership.
- Add compact bilingual pages for the highest-traffic topics.
- Add provider-neutral examples for streaming, tool authorization, and durable execution.
- Add a small public evaluation dataset with versioned annotations.

## Later

- Publish role-specific mock interview packs for platform, application, and research agents.
- Add optional integrations with local models and popular tracing backends.
- Compare agent frameworks by primitives and operational behavior rather than popularity.
- Build a contributor-maintained index of anonymized interview themes with freshness dates.
