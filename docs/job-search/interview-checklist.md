# Interview Checklist

- [ ] Two-minute introduction and project story.
- [ ] One coding challenge runnable from a clean checkout.
- [ ] One system design diagram with SLOs and trade-offs.
- [ ] Five behavioral stories with measurable outcomes.
- [ ] Examples of evaluation, observability, security, cost, and reliability.
- [ ] Questions about data access, on-call, deployment, and model/provider policy.
