# Mock Interviews

Use a timer and ask a partner to interrupt with “why?”, “what fails?”, and “how do you know?”. The goal is not to recite model terminology; it is to make assumptions, boundaries, evidence, and trade-offs legible.

- [Junior Agent Engineer](junior-agent-engineer.md)
- [Senior Agent Engineer](senior-agent-engineer.md)
- [Agent system design](agent-system-design.md)

After each session, record one unclear answer, one missing test, and one question you would ask the team.
