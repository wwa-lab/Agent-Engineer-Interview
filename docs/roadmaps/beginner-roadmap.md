# Beginner Roadmap

Learn the vocabulary, build one bounded loop, and practice saying what happens when the happy path fails.

1. Read [LLM basics](../fundamentals/llm-basics.md), [structured output](../fundamentals/structured-output.md), and [function calling](../fundamentals/function-calling.md).
2. Study [what is an agent?](../agent-engineering/what-is-an-agent.md) and [workflows vs agents](../agent-engineering/workflows-vs-agents.md).
3. Complete [Challenge 1](../../coding-challenges/01-tool-calling-agent/README.md).
4. Learn RAG with [the overview](../rag/rag-overview.md) and complete Challenge 2.
5. Practice [beginner questions](../interview-questions/beginner.md) out loud.

Graduation evidence: a small README with an architecture diagram, five tests, one failure demo, and a two-minute explanation.
