# Intermediate Roadmap

Assume you can build a normal API and want to demonstrate agent-specific judgment.

1. Compare chunking, retrieval, reranking, and hybrid search.
2. Add session/long-term memory with deletion and access boundaries.
3. Study MCP, evaluation, tracing, cost, reliability, and prompt injection.
4. Complete two challenges and one [system design case](../system-design/system-design-framework.md).
5. Practice [intermediate questions](../interview-questions/intermediate.md) with metrics.

Graduation evidence: a versioned evaluation set, a trace schema, a cost/latency budget, and a failure taxonomy.
