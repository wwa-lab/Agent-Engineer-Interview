# LLM Basics

An LLM predicts tokens from context. For interviews, connect the abstraction to engineering: token limits affect cost and latency, sampling affects variance, and model knowledge can be stale or unsupported. A model is one component in a system with retrieval, tools, validation, and policy.

Practice explaining what the model knows from its parameters, what the application supplies at request time, and what must be checked outside the model. See [tokens, context, and embeddings](tokens-context-and-embeddings.md).
