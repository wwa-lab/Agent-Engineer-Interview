# Rapid Review

Use these one-line prompts for the final day. A complete answer still needs one example and one failure mode.

- What is the smallest action that needs autonomy?
- Which tool call is unsafe to retry?
- What evidence supports a RAG answer?
- How can memory leak tenant data?
- What does an evaluator miss?
- Which metric moves when cost is optimized?
- How do you detect provider drift?
- Where does approval happen?
- What is your no-answer UX?
- What would you remove if the budget were cut in half?
