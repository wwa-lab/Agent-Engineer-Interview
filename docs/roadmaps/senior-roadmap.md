# Senior Roadmap

Focus on operating boundaries, not framework breadth.

1. Design a platform or high-impact workflow with explicit autonomy tiers.
2. Define evaluation ownership, release gates, provider migration, and incident response.
3. Practice [senior questions](../interview-questions/senior.md) and all five system design prompts.
4. Write a [project story](../job-search/project-storytelling.md) that includes failure and measurable improvement.
5. Review security, tenancy, memory governance, and long-running tasks.

Graduation evidence: you can describe a safe launch plan, rollback, and what the platform will refuse to guarantee.
