# Code of Conduct

We want this project to be a welcoming place for careful technical discussion. Participants must help create an environment free from harassment, discrimination, personal attacks, doxxing, sexualized conduct, and deliberate disruption.

Use evidence and curiosity when disagreeing. Critique ideas and code, not people. Assume good intent while taking impact seriously. Maintainers may remove comments, close contributions, or restrict participation when behavior violates this standard.

Report a concern privately to the repository maintainers through the contact method configured in the GitHub repository. Please include links, dates, and a concise description; do not publish sensitive details in a public issue.
