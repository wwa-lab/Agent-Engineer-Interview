# Junior Agent Engineer Mock Interview

## 45-minute format

1. **5 min:** Explain a project or a small agent loop.
2. **15 min:** Answer three beginner questions: structured output, RAG, and tool calling.
3. **15 min:** Implement a bounded tool dispatcher from [Challenge 1](../coding-challenges/01-tool-calling-agent/README.md).
4. **10 min:** Explain a failure: malformed arguments, empty retrieval, or a model outage.

## Evaluation rubric

Look for clear types, input validation, a stop condition, readable tests, and willingness to say “I would verify that.” Penalize unsafe assumptions more than missing framework vocabulary.
